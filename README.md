# CleanHouse

Frontend-only company profile website for a professional cleaning service business based in Magelang, with service coverage in Magelang, Yogyakarta, Jakarta, and Bali.

## Stack

- React
- React Router
- Tailwind CSS v4
- lucide-react
- Vite

## Features

- Six pages: Home, Services, Pricing, Gallery, About, Contact
- Static data modules for navigation, services, pricing, gallery, FAQ, and service areas
- WhatsApp booking and quote CTAs with pre-filled messages
- Frontend-only quote request form
- Local gallery category filtering
- Responsive navbar with mobile menu
- No backend, database, auth, admin dashboard, booking engine, or payment system

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
